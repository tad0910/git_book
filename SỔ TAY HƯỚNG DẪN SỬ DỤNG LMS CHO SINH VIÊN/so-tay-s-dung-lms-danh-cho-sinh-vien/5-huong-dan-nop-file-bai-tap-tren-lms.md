# (5) HƯỚNG DẪN NỘP FILE BÀI TẬP TRÊN LMS

## I. Mục đích

* Nộp file bài tập dễ dàng theo từng môn học, từng tuần học

#### Bước 1: Tại khóa học cần nộp bài tập, chọn Assignments, chọn bài tập cần nộp
